Состояния модели

  • ng-untouched - если элемент ввода еще не получал фокус
  • ng-touched - если поле ввода уже получало фокус
  • ng-invalid - если значение некорректно
  • ng-valid - если значение в поле ввода корректно
  • ng-pristine - первоначальное значение не изменялось с момента загрузки страницы
  • ng-dirty - первоначальное значение в поле ввода было изменено
  • ng-pending - ожидание асинхронной валидации
